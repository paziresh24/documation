# نصب SDK

<Steps>

 ### افزودن SDK به فرانت‌اند ابزارک
اسکریپت زیر را در تگ `<head>` صفحه خود قرار دهید:
```html
<script src="https://hamdast.paziresh24.com/sdk/hamdast.js"></script>
```
 
### آماده‌سازی SDK
 سپس برای آماده‌سازی آن را initialize کنید.
 ```js
window.hamdast.initialize({ app_key: "YOUR_APP_KEY" });
```
</Steps>
